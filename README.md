##  PrivateSea 🌊


### **Project Info 💎** ###
NFT 거래와 동시에 NFT를 생성, 저장을 할 수 있는 NFT 마켓플레이스 OpenSea 클론코딩 Project.

---

### **Achievement Goals 🗝** ###
* 메타마스크 지갑연결
* 로그아웃 기능 구현
* NFT 민팅 페이지 구현
* NFT 거래 
* 테마별 NFT 조회 페이지
* 마이페이지에서 나의 정보, 나의 NFT 확인
